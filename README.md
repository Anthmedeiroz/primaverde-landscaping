# Primaverde Landscaping — Website

Marketing site for Primaverde Landscaping. Single static page (`index.html`) styled with Tailwind (CDN) and vanilla JS — no build step.

**Live:** https://primaverdelandscaping.com

## Structure
- `index.html` — the entire site (styles + scripts inline)
- `PV Logo Light.png` — nav/footer logo
- `PV Logo T.png` — favicon
- `vercel.json` — Vercel static config (clean URLs)

## Local development
```bash
npm install      # installs puppeteer (used only for screenshots, optional)
node serve.mjs   # serves at http://localhost:4137
```

## Deploying
Hosted on Vercel. Pushing to the default branch on GitHub auto-deploys once the repo is connected to the Vercel project.
